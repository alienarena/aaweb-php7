Debian and Ubuntu

Alien Arena 7.71.3 will be released soon in Debian 12 - Bookworm and Ubuntu 23.04 - Lunar Lobster!
The release date for Debian 12 is not known yet. The release date for Ubuntu 23.04 is April 20th, 2023.

New release 7.71.4

Release 7.71.4 has most game operations running on alienarena.org. Also the old obsolete stats have been removed.

New IRC server

For IRC, use irc.alienarena.org. For secure connections, use port 6697, otherwise 6667.

How to install

There are multiple options to install this game:

  • Install from SVN (Windows/Linux). You'll have the latest code (see below).
  • Install from the Snap Store (Linux only).
  • Install from Steam (Windows/Linux), but it is outdated.
  • Install the obsolete demo (Windows/Linux). The demo is not being maintained.

SVN

The repository is located at: svn://svn.icculus.org/alienarena/trunk.
On Windows, you can install TortoiseSVN and download the source code. The alienarena.exe is already there.
On Linux, install subversion. Then you need to build the application yourself.

If you have already done this before, use svn update to download the latest sources.
Optionally, you can specify the number of threads to run make with, like make -j 4.

Other configure options that are useful:
Dedicated server: ./configure --disable-client
With debug symbols: ./configure --enable-debug
